___Элементы интерфейса___

Поле «Порт»
Выпадающий список для выбора порта, к которому отправляются команды

Поле «Расстояние, мм»
Числовое поле, задаёт величину перемещения датчика за одну команду.

Переключатель «Вперед (+)»
Выбирает движение датчика в положительном направлении от нуля.

Переключатель «Назад (-)»
Выбирает движение датчика в отрицательном направлении от нуля.

Быстрые кнопки расстояний (1, 5, 10, 20)
При нажатии подставляют своё значение в поле «Расстояние, мм».

Кнопка «Двигать»
Отправляет команду на перемещение датчика на указанное расстояние с выбранным направлением.

Кнопка «Тест»
Включает тестовый режим программы без подключения к порту, чтобы продемонстрировать работу программы.

Сборка .exe
Для сборки автономного приложения используется PyInstaller
