_Элементы интерфейса_

Поле «Порт»
Выпадающий список для выбора датчика, к которому отправляются команды.

Поле «Журнал»
Список последних изменений.

Переключатель «Вперед»
Выбирает движение датчика в положительном направлении от нуля.

Переключатель «Назад»
Выбирает движение датчика в отрицательном направлении от нуля.

Быстрые кнопки расстояний (1, 5, 10, 20)
При нажатии подставляют своё значение в поле «Расстояние, cм».

Кнопка «Очистить»
Стирает всю историю журнала. 

Кнопка «Двигать»
Отправляет команду на перемещение датчика на указанное расстояние с выбранным направлением.

Кнопка «Проверка»
Позволяет проверить без подключения.
